# To install dependencies:

```sh
bun install --frozen-lockfile
```

# To run:

```sh
bun run dev
```

open http://localhost:3000

# To test:

```sh
bun test

# or

bun test path/to/testfile.tsx
```

# To test with coverage:

```sh
bun test --coverage

# or

bun test path/to/testfile.tsx --coverage
```
